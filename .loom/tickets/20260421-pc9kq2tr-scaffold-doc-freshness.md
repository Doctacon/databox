# Goal

Replace every stale `task check:layout` / `task staging:generate` reference with
the direct `python scripts/...` form. Those Taskfile targets were removed in
`ticket:taskfile-trim` but several strings still point at them, including strings
generated by the scaffold.

# Why

Friction point #3 in `ticket:add-usgs-earthquakes-source`. A new operator runs the
command printed in `new_source.py`'s "next steps" message and gets
`task: No matches found for "check:layout"`. Same thing for the header comment on
every generated `stg_*.sql`. Confidence-killing on day one.
